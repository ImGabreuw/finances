Funcionalidades

Fornecer dados para outras APIs de um determinado FII:

  • Cotação
    • Fornecer as informações da cotação (atualizar a cada 5 min)
  • Indicadores
    • Fornecer os indicadores do ativo (atualizar a cada semana)
  • Informações gerais
    • Fornecer as informações gerais do ativo (atualizar a cada semana):
  • Eventos corporativos
    • Dividendos
      • Fornecer os dados do último dividendo
      • Obter próximo dividendo
      • Obter o histórico de dividendos
        • Obter os dividendos pagos no ano anterior
        • Obter os dividendos pagos no ano atual
        • Fornecer a comparação dos dividendos pagos entre o ano atual e o anterior
    • Subscrição
      • Obter os dados da última subscrição
  • Portfólio
    • Imóveis
      • Listagem de todos os imóveis de um ativo
    • FIIs
      • Listagem dos FIIs do portfolio de um ativo
  • Notícias
    • Listagem de todas as notícias de um ativo
    • Obter notícias a partir de palavras-chave do titulo ou o título completo
    • Listagem das notícias com data de publicação entre um intervalo de datas
  • Comunicados
    • Listagem de todos os comunicados de um ativo
    • Obter o último comunicado de um ativo
    • Obter comunicados a partir de palavras-chave do título ou o título completo
    • Obter comunicados com data de publicação entre um intervalo de datas de um ativo

Casos de uso

  • Obter a cotação do dia
    1. Envio da requisição para a API do Finaces Data Provider:
      GET http://localhost:3000/real_state_fund/quotation/{assetCode}
    1. Execução do processo de extração de dados
      1. Recuperar a cotação no banco de dados via propriedade codigoDoAtivo
      1. Se a diferença da data e hora atual e o valor da propriedade dataEHoraBase for menor do que 5 min, então retornar essa cotação
      1. Caso contrário extrair os dados da cotação no Google Finances
        1. Definir a propriedade dataEHoraBase com o valor da data e hora atual
        1. Tratar os dados coletados no Google Finances
        1. Salvá-los no banco de dados
    1. Envio de um resposta para o cliente no seguinte formato:
      {
	"código do ativo": ...,
	"data base": "yyyy-MM-dd",
	"preço atual": ...,
	"variação atual em porcentagem": ...,
}

  • Obter último dividendo pago
    1. Envio da requisição para a API do Finaces Data Provider:
      GET http://localhost:3000/real_state_fund/corporative_events/dividends/last_dividend/{assetCode}
    1. Execução do processo de extração de dados
      1. Recuperar os dados no banco de dados via propriedade codigoDoAtivo e ultimoDividendo == true
      1. Se a diferença da data atual e o valor da propriedade dataBase for menor do que 1 semana, então retornar os dados do banco de dados
      1. Caso contrário extrair os dados do último dividendo do ativo no Status Invest

        Verificar se o os dados coletados no Status Invest são iguais do banco de dados

        1. Se sim:
          1. Atualizar a propriedade dataBase com o valor da data atual
          1. Retornar os dados do banco de dados
        1. Se não:
          1. Definir a propriedade dataBase com o valor da data atual
          1. Definir a propriedade ultimoDividendo com o valor true
          1. Definir a propriedade proximoDividendo com o valor false
          1. Tratar os dados coletados no Status Invest
          1. Salvá-los no banco de dados

    1. Envio de uma resposta para o cliente no seguinte formato:
      {
	"código do ativo": "...",
	"data base": "yyyy-MM-dd",
	"data com": "yyyy-MM-dd",
	"data de pagamento": "yyyy-MM-dd",
	"valor": ...,
}
